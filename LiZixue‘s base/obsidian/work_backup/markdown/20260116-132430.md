### 金属

| 材料       | 特性                                        | 应用             |
| -------- | ----------------------------------------- | -------------- |
| **Q235** | 焊接性能好，易生锈加工处理完需要喷粉处理                      | 设备上面的焊接底架或者焊接件 |
| **45钢**  | 强度高，重量比较重，容易生锈需要镀铬处理                      | 有强度要求的场合       |
| **不锈钢**  | 强度高，重量比较重，不易生锈，比较硬不容易机加工处理                | 有强度要求的场合，钣金件   |
| **铝合金**  | 价格便宜，重量轻，机加工性好，不易生锈一般通过阳极氧化、喷砂白色、喷砂黑色美化处理 | 型材护罩，没有强度要求的场合 |

### 非金属

| 材料             | 特性                | 应用                                |
| -------------- | ----------------- | --------------------------------- |
| 大理石            | 硬度高，延展性几乎没有       | 适合高精度精密加工，精密测量类设备、高速运行的平台         |
| **PU聚氨酯**      | 耐油性优良、耐磨性非常好      | 适合做缓冲垫、减震头、皮带辊、滚筒                 |
| 玻璃纤维           | 抗腐蚀、绝缘、耐高温        |                                   |
| **硅胶**         | 无毒无味、绝缘           | 医疗行业、食品行业、密封圈、硅胶保护，防止接触产品太硬导致产品磨损 |
| 聚四氟乙烯（铁氟龙、特氟龙） | 粘连性差、耐酸碱、摩擦系数很小   | 不粘锅、电饭锅内胆。                        |
| **亚克力**        | 透光                | 非标设备美观、安全门                        |
| PVC（聚氯乙烯）      | 价格便宜              | 皮带。                               |
| PE（聚乙烯）        |                   | 皮带、医学领域。                          |
| **POM（赛钢）**    | 硬度和耐磨性好           | 可直接做机加件                           |
| **PC板**        | 机加工性能，精度高，可抛光使性能好 | 制作模板                              |
| **Peek材料**     | 具有防静电             | 如果需要直接接触产品，常用Peek                 |
