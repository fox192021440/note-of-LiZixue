title:
  - 常见表面处理选用推荐
tags:
  - 非标设计
---

| 材料                            | 表面处理                   | 特点                            | 备注                    |
| ----------------------------- | ---------------------- | ----------------------------- | --------------------- |
| 铝合金                           | 喷砂黑色、喷砂白色              | 喷砂工艺制造成本相对较低                  | 喷砂后平面度变化不大，表面粗糙度3.2左右 |
|                               | 亮黑（氧化）、阳极本色            | 表面光洁度好、容易反光                   | 前后平面度变化不大，表面粗糙度1.6左右  |
|                               | 金色硬阳、硬质阳极（黑色）、硬质阳极（本色） | 7075铝合金硬阳处理，批次不同，，材料含铜量差异会有色差 | 防静电性能稳定，防划性合格         |
| 45钢，SKD61，Cr12                | 镀铬                     | 表面光洁度好，尺寸变化小                  | 推荐使用                  |
|                               | 镀黑铬                    | 表面光洁度好，尺寸变化小                  | 可替代发黑处理，颜色有要求使用       |
|                               | 发黑                     | 尺寸变化小，薄膜太薄易生锈，价格便宜            | 防锈能力差，不推荐             |
| 不锈钢系列（304、440C、40Cr、136、201等） | 不做表面处理                 |                               |                       |
| 不锈钢201板                       | 陈本排序：不做表面处理>拉丝>单镜面>双镜面 |                               |                       |
| 镀锌板                           |                        |                               |                       |
| Q235焊接件，方通                    | 喷粉处理                   | 象牙白、电脑白、米黄色、亮黑等               | 焊接件不建议镀铬，推荐喷粉         |
| 非金属材料（有机玻璃、PC、PVC、石英玻璃等）      | 抛光工艺、镀防静电膜             |                               |                       |
