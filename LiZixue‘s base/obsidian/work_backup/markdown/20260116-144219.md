机械加工工艺主要包括手动加工：
比如（车、铣、刨、磨、电火花、激光切割等等）还有数控加工

另外一种常见的加工工艺是焊接工艺。

| 加工方式       | 应用                            |
| ---------- | ----------------------------- |
| 车削         | 加工轴、盘、套和其他具有回转表面的工件           |
| 铣削加工       | 可以加工平面、沟槽、轮廓，也可以加工各种曲面        |
| 刨削加工       | 主要用于零件外形加工                    |
| 磨削加工       |                               |
| 电火花加工（线切割） | 能加工普通切削加工方法难以切削的材料和复杂形状工件     |
| 激光切割       | 切割4mm以下不锈钢，激光束中加入氧气可切割20mm的碳钢 |
| 数控机床       | 能够加工复杂零件的高效率自动化机床             |
