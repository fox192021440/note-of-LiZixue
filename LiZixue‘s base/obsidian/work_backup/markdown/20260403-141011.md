### 需求

- FC 后端作业要求：Magazine 料盒内产品片数必须为双数。

- 实际生产中：每批产品最后一个料盒存在单数情况。

- 当前做法：依赖人工临时处理，一致性与效率不足,且不利于后续全线自动化推进。

- 因此需要：一套专用凑双辅助设备，实现标准化、自动化处理。


### 设备功能

- 存放不同机种对应的样品假片，在需要时，补入单数料盒，实现双数作业。
- 与 MES 判定逻辑联动不改变现有 FC 主制程节拍。
- 使用频率低优先保证简单、稳定、易维护
- 料盒搬运
- 假片补入动作
- 与 MES 作业逻辑配合
- 系统判别是否需要补料操作

### 方案

| 流程方向             | 步骤  | 核心动作         | 目的                                 |
| ---------------- | --- | ------------ | ---------------------------------- |
| **正向取料（料盒→皮带线）** | 1   | 天车下料         | 上游天车（OHT）将满载产品的 Magazine 料盒送到设备下料口 |
|                  | 2   | 取料机械臂取料盒     | 取料机械手从下料口抓取指定料盒                    |
|                  | 3   | 放对接工作台 + 拆挡杆 | 料盒定位，拆除料盒挡料杆（防止产品掉落的结构）            |
|                  | 4   | 取料机械臂后推产品    | 从料盒后方将产品向前推出                       |
|                  | 5   | 接料机械臂接料      | 在料盒前方接住被推出的产品                      |
|                  | 6   | 皮带流线取料       | 皮带正转，将产品完全从料盒带出，输送到下游工序            |
| **反向回料（皮带线→料盒）** | 7   | 接料机械臂回工作台    | 接料机械手回到对接工位，准备回料                   |
|                  | 8   | 皮带反转回料       | 皮带反向运转，将产品送回料盒                     |
|                  | 9   | 推手补推         | 推手下降，将未完全推入料盒的产品推到位                |
|                  | 10  | 取料机械臂送回料盒    | 将装满 / 空料盒送回天车下料口，完成循环              |

取料机械手动作：1.天车下料位置取料盒；2.工作平台位置拆挡杆；3.可调节存储位推假片；4.回工作台接假片，关挡杆；5.回下料位还料盒。

接料机械手动作：1.可调节存储位置接假片，皮带线取假片；2.工作台位送假片进取料机械手上料盒。

#### 取料机械手

功能分工明确的复合抓手设计：前端约 50 mm 低厚度（3mm)结构
→ 用于从假片料盒中推出产品

后端加厚(10mm)结构
→ 用于稳定抓取magazine料盒

单一机构完成：料盒搬运、假片推出；降低机构数量与控制复杂度

![[img/20260403-142043.png|500]]

### 可调存储位

三层分层结构，每个储位设可调定位柱，支持前后、左右调节并锁紧固定，兼容四种尺寸料盒。人员可根据排产需求快速调整，实现多规格共用与高效换型。
因前后受机械手臂空间限制，料架底部配置重载滑轨，可整体侧向抽出进行调节与取放料；作业完成后推回锁定，兼顾操作便利性与运行稳定性。

![[img/20260403-142442.png|500]]

#### 拆挡杆机构

复合定位与执行机构：
可移动式双定位块结构
→ 用于兼容并固定四种不同尺寸料盒的挡杆

集成于直线模组的拆档杆机构
→ 用于精确压入并旋开挡杆

单一模组实现：​ 多尺寸适应、挡杆稳定打开；简化了料盒对接流程，提升了系统可靠性
