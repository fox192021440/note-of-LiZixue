| 名称           | 使用温度区间                 | 化学性质                        | 物理性质                                                 |
| ------------ | ---------------------- | --------------------------- | ---------------------------------------------------- |
| FFKM（全氟橡胶）   | -20℃~320℃（短期可达 350℃+）  | 几乎耐受所有化学介质（强酸、强碱、酮、胺、高温蒸汽等） | 高温下弹性优异，压缩永久变形极小（价格极高，为 FKM 的 5-10 倍、EPDM 的 20-50 倍） |
| FKM（氟橡胶）     | -20℃~200℃（特殊配方短期 250℃） | 耐油（燃油、液压油）、有机溶剂，不耐酮类、胺类     | 高温下压缩永久变形小（优于 EPDM），价格较高（为 EPDM 的 3-4 倍）             |
| EPDM（三元乙丙橡胶） | -50℃~150℃              | 耐水、蒸汽、酸碱，不耐矿物油、汽油           | 低温弹性极佳，高温下压缩变形增大，价格低（性价比突出）                          |
| 丁晴橡胶         | -50℃~120℃              | 耐水、蒸汽、酸碱                    | 价格低                                                  |
