---
title: 基金止盈&止损比例设定
tags:
---


| 基金类型         | 止盈比例    | 止亏比例       | 逻辑说明                 |     |
| ------------ | ------- | ---------- | -------------------- | --- |
| 股票型 / 行业主题基金 | 15%-25% | 15%-20%    | 波动大，止盈留一定弹性，止损控制风险   |     |
| 混合型基金（偏股）    | 10%-20% | 10%-15%    | 股债平衡，波动比纯股票型小，比例适当降低 |     |
| 混合型基金（偏债）    | 5%-10%  | 5%-8%      | 收益稳健，止盈不用等太高，止损概率低   |     |
| 纯债基金         | 3%-6%   | 不设止损（除非踩雷） | 年化收益低，止盈对应年化，下跌风险极小  |     |
