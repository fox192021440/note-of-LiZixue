- [ ] 将SAI500带差速锁客户来源数模转为solidworks格式。
- [ ] 对SAI500数模按照每个工位按照什么零件分类存放在一个文件夹中。

6095和6160装油冷器，少一个油冷器，可削减一个工位。
3110和3120装电子油泵，3110少一个电子油泵，螺钉数量削减两颗。

- [ ] 按照外协一群负责工位输出对应工位数模。
- [ ] 将一群工位数模转为step格式。
- [ ] 将这些数模发给一群并说明sai500数模跟新情况。

- [ ] 按照外协二群负责工位输出对应工位数模。
- [ ] 将二群工位数模转为step格式。
- [ ] 将这些数模发给二群并说明sai500数模跟新情况。

- [ ] 按照外协三群负责工位输出对应工位数模。
- [ ] 将三群工位数模转为step格式。
- [ ] 将这些数模发给三群并说明sai500数模跟新情况。

- [ ] 按照外协四群负责工位输出对应工位数模。
- [ ] 将四群工位数模转为step格式。
- [ ] 将这些数模发给四群并说明sai500数模跟新情况。

- [ ] 按照外协五群负责工位输出对应工位数模。
- [ ] 将五群工位数模转为step格式。
- [ ] 将这些数模发给五群并说明sai500数模跟新情况。


- [ ] 按照测量设备群负责工位输出对应工位数模。
- [ ] 将测量设备群工位数模转为step格式。
- [ ] 将这些数模发给测量设备群并说明sai500数模跟新情况。要求测量设备群人员设计设备时按照带有差速锁的数模进行，并预留不带差速锁的数模壳体测量设备。
