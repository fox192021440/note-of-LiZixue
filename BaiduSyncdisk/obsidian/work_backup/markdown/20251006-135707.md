4群负责工位
3049，3050，5099，6010

- [x] 3049转子上料方式（滑台or机器人）
- [x] 如果滑台上线，需要转子上线滑台高度及数模 
- [x] 两个机器人从哪里把两个工件抓入机器 

人工通过kbk吊具将转子放在工装台上，工装台经过滑台进入行架，行架将转子转运至机械手（左1）抓取位置，机械手转运转子进入转子总成入壳工站。
机械手（左2）将转子总成入壳工站中的工件抓取进入定转子合装工位。
以上就是转子上料方式。
![[img/20251006-154451.png|300]]

机械手（右1）从定子热套线抓取工件进入定转子合装工位。
![[img/20251006-154604.png|300]]

- [x] 定转子合装下线位置

机械手（左2）将合装好的工件抓取至合箱螺栓预紧的线首位置。
![[img/20251006-155018.png|300]]

- [x] 机械手快换夹放的位置 ✅ 2025-10-09
- [x] 3051是哪个外协在设计（公用三套快换夹爪），快换夹供应商群 ✅ 2025-10-09
- [x] 垫片厚度公差范围，垫片规格有多少种类，每种上面放多少个2025-10-06

先是中壳由壳体测量工位被机械手（下1）抓取进入选垫压装工位。
选垫上料工位属于手动工位，人工选择合适的垫片放在转台上，转台转动180°，机械手（下1）抓取垫片放入压装工位进行压装。
![[img/20251006-160345.png|300]]
谭工只需要设计这个转台就可以了，所以不需要提供垫片规格参数。
![[img/20251006-160433.png|300]]

- [x] 齿轮箱合装上料位置
- [x] 齿轮箱合装中壳上料位置

机械手（上1）将选垫压装好的中壳由等离子清洗工位转运至齿轮箱合箱翻转位置。
机械手（上1）将齿轮箱由轴系分装区线体转运至齿轮箱合箱位置。
![[img/20251006-160528.png|300]]

- [x] 齿轮箱合装齿轮箱上料位置和下线位置需要放置工位

存在工位包含顶升机构。
