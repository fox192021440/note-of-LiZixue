### 气缸类型

| 类型    | 特性                            |
| ----- | ----------------------------- |
| 单作用气缸 | 气缸单个方向由压缩空气驱动，相反方向依靠弹簧复位。     |
|       | 行程较短，100mm以内。                 |
|       | 小巧简单、易维护，经济实惠。                |
|       | 功能单一，适合简单工况。                  |
| 双作用气缸 | 气缸活塞两个方向运动都有压缩空气驱动。           |
|       | 可以控制推拉动作、准确性高。                |
|       | 适合复杂工况，比如两个方向都需要输出力、需要控制直线运动。 |
|       | 成本较高。                         |

| 类型      | 特性                           |
| ------- | ---------------------------- |
| 标准筒形气缸  | 制造成本低，价格便宜。                  |
|         | 需要搭配**浮动接头**，占用空间大。          |
| 自由安装型气缸 | 灵活方便，占用空间小。                  |
| 薄型气缸    | 负载移动距离小，节省安装空间。              |
|         | 相同行程条件下有最短长度气缸，体积小、重量轻、安装方便。 |
|         | 常用于Z轴短程升降，或设备上施加组装、贴合力的工况。   |
| 摆动气缸    | 体积小、安装方便、摆动角度可调。             |
|         | 只能来回摆动旋转，调节好之后固定角度。          |
| 带导杆气缸   | 负载移动需要高精度导向，气缸需要较高的抗扭转刚度。    |
| 双联气缸    | 采用双气缸设计，同样的缸径可具备两倍输出力。       |
|         | 不回转精度高、侧面负载抵抗力强。             |


### 气缸选型

负载率$\eta$是气缸活塞杆上承受轴向负载力与气缸理论输出力之比。负载率$\eta$的大小选取，与气缸动作状态有关。
静载荷，$\eta=70\%$;
气缸速度在50-500mm/s的水平或垂直动作，$\eta=50\%$;
气缸速度在500mm/s以上，$\eta=30\%$。

在某自动夹紧场合，气缸在水平方向运动，要求气缸夹紧力为300N，气缸工作时空气压力为0.6MPa，试以费斯托公司气缸为例选定合适系类缸径。

- 计算所需要的气缸理论输出力,夹紧工件属于静载荷，安装$\eta=70\%$进行计算：
$$F_0 =\frac{F*100\%}{\eta}=\frac{300*100\%}{70\%}=428.6N=43.7kgf$$
- 在0.6MPa工作压力下，缸径为25mm的气缸水平方向工作时，输出理论力为29.4kgf；缸径为32mm的气缸理论输出力为48.3kgf。因此玄策缸径为32mm的才能满足需求。

### 气缸安装组合方式

气缸推动的对象两端需要设置液压缓冲器或者限位螺栓。
两者可同时使用也可以单独使用限位螺栓

每台机要在接气源处接**三联件**，三联件包含**过滤器（过滤空气杂质）、减压阀（维持气压恒定）、油雾器（提供润滑油）、压力表（气压指示仪器）**。

通过**节流阀**调节气流量大小，控制气缸的速度。

通过**电磁换向阀**控制气缸的往复运动、真空吸盘的吸取和放置动作。
